# 🤝 Partner Program — “Заклятие знаний”

This document describes the official partner (affiliate) program for the “Заклятие знаний” ecosystem.

The goal of the program is to distribute the system through independent agents who help explain, demonstrate, and grow adoption of the open-source core and paid learning layer.

---

## 🧠 What is the Partner Program?

The partner program is a distribution layer built around the project.

A partner is not just a reseller — they are a **system interpreter** who:
- explains the idea of the system
- shares the open-source repository
- drives participation in the master-class
- builds their own audience around the methodology

---

## 🔗 What you promote

Partners can promote:

### 1. Open-source core
- GitHub repository (BioNum / system core)
- basic documentation
- examples and use cases

### 2. Master-class
- “Заклятие знаний” activation session
- live breakdown of the system
- practical application workflows

### 3. Future extensions (optional)
- advanced modules
- private engine versions (v2+)
- consulting sessions

---

## 💰 Commission Structure

### Base commission
- 30% per sale of the master-class

### Performance tiers

#### Level 1 — Starter
- 30% commission
- access to referral links and basic assets

#### Level 2 — Builder
(10+ confirmed sales)
- 40% commission
- access to marketing scripts and case studies

#### Level 3 — Architect
(50+ sales or consistent traffic)
- 50% commission
- early access to new versions
- ability to build derivative “branches” of the system

---

## 🧩 How the funnel works

Partners typically guide users through 3 stages:

### 1. Awareness
- posts about the concept of the system
- explanation of the open-source idea

### 2. Interest
- demonstration of the GitHub repository
- showcasing how the system interprets language / knowledge

### 3. Conversion
- invitation to the master-class
- explanation of what is unlocked inside the session

---

## 📦 Partner kit includes

Each approved partner receives:

- referral link
- official landing page copy
- description of the system (short + long versions)
- 5–10 ready-to-use posts
- FAQ for audience questions

---

## ⚙️ Rules

### Allowed
- publishing content about the system
- personal interpretations and explanations
- building educational content around the idea
- running organic promotion (posts, videos, threads)

### Not allowed
- spam or mass unsolicited messaging
- misleading income promises
- misrepresentation of official product status
- pretending to be the core author or maintainer

---

## 📊 Tracking & payouts

Tracking is handled via:
- referral links with unique IDs
- UTM parameters (optional)
- affiliate dashboard or manual reporting system (initial phase)

Payouts are processed periodically based on platform rules (weekly or monthly depending on infrastructure).

---

## 🧠 Philosophy of the network

This program is not traditional affiliate marketing.

It is based on the idea that:

> A system grows when it is reinterpreted, not just advertised.

Each partner is expected to adapt the message to their own audience while preserving the core idea of the system.
